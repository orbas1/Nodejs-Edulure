# Navigation & Menu Requirements

## Global Navigation
- Logo → Home → Explorer → Communities → Dashboard → Assets → Settings.
- Right cluster: Search, Notifications, Create, Profile menu.
- Provide theme toggle within profile dropdown.

## Secondary Menus
- Tab bars on dashboards (Overview, Communities, Assets, Monetisation).
- Sub-navigation within settings for categories.
- Breadcrumbs for deeper routes (Course detail > Modules > Lesson).

## Contextual Menus
- Kebab menus on cards for quick actions (edit, duplicate, share).
- Bulk action toolbar appears when items selected.

## Mobile Navigation
- Bottom nav with Home, Explorer, Communities, Notifications, Profile.
- Create button as floating action near center.
